Loops in JS

While loop

In Javascript, a While loop is a control flow statement that allows you to repetadely execute a block of code as a specified condition is true.

Do While loop

In Javascript, a do-while loop is a control flow statement that repatedly excutes a block of code as long as a specificed condition evalutes to true.

For Loop

For Loop runs through the block of code a predetermined number of times.And it works well when you know exactly how many iterations you need.

types of for loop

1.for/in- loop through the properties of an object
2.for/of- loop through the properties of array and string
